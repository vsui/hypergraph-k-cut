# Hypergraph partitioning algorithms

## Building

```
mkdir build
cd build
make
```

## Usage

To generate hypergraphs, run `./gen <num_vertices> <num_edges> <sampling percentage>`

To run the algorithm, run `./main <input file> <k> <algorithm>`

## References
